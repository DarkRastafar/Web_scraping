README

get_pages() - получает на вход базовый URL, примешивает к нему пагинацию и сохраняет в папку pages

get_data() - открывает на чтение скачанную страницу и достает из нее необходимые данные. Ссылки пишутся в
pages/data, а собранные данные в json в data_result.

facad_parser() - агрегирующая функция
